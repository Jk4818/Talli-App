# **App Name**: Splitzy

## Core Features:

- Landing Page: Landing page with 'Split Now' and demo options, clear visual hero section.
- Demo Mode: Demo mode with prepopulated mock data to demonstrate app functionality.
- Participant Management: Participant management: Add and remove participants for bill splitting.
- AI Receipt Processing: AI-powered item extraction: Automatically extracts items, costs, discounts, and service charges from receipt images, using a tool to decide when to flag for manual review.
- Receipt Management: Receipt management: Upload, edit receipt details, and assign payers.
- Item Management: Item list editor: Filter, sort, edit item details and flag items.
- Item Assignment: Item assignment: Assign items to participants via a swipeable interface, with equal splitting logic and penny-perfect display.
- Summary and Settlement: Bill breakdown and settlement: Multi-currency handling and penny-perfect calculation.

## Style Guidelines:

- Primary color: Saturated purple (#A020F0) to convey a sense of modernity and financial technology, with a subtle nod to reliability and trustworthiness.
- Background color: Very light desaturated purple (#F0E5F7) to provide a clean and unobtrusive backdrop, ensuring readability and focus on the content.
- Accent color: Blue (#20A0F0), a slightly brighter hue to offer good contrast, for interactive elements like buttons and highlights, drawing the user's attention without overwhelming the interface.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use a consistent set of minimalist icons from a set like Lucide or Remix Icon to visually represent receipt categories and user actions.
- Implement a responsive, card-based design leveraging Tailwind CSS, ensuring seamless adaptation across devices. Employ clear visual hierarchy.
- Employ subtle animations using Framer Motion for UI transitions and user interactions, such as item assignment confirmations.
